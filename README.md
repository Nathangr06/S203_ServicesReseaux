# S203_ServicesReseaux

Les objectifs de cette SAE 2.03 concernent principalement la mise en place d'un réseau de machines virtuelles (VM) et la configuration de plusieurs services réseau :

Réseau de VM :

Mettre en place un réseau local (LAN) avec 3 VMs sur le même PC ou sur plusieurs PC via le gestionnaire de machines virtuelles QAMU.

Configurer ces VMs pour qu'elles puissent communiquer entre elles.

Serveur Web :

Installer et configurer un serveur Apache.

Héberger un site web statique (développé au S1).

Configurer un « ServerName » personnalisé.

Configurer le serveur DNS pour gérer ce nom de domaine.

Ajouter des options avancées comme l'authentification par mot de passe et le contrôle d'accès.

Serveur de messagerie :

Installer et configurer un serveur de messagerie avec Postfix et Dovecot.

Créer des comptes de messagerie pour les tests.

Tester l'envoi et la réception des mails via Thunderbird.

Serveur DNS :

Configurer un serveur DNS primaire pour gérer la résolution des noms de domaine du site web.

Ajouter un serveur DNS secondaire.

Définir plusieurs noms de machines dans la zone DNS (ex : www, dns1, dns2, smtp, imap).

Modifier /etc/resolv.conf pour définir ce serveur comme DNS par défaut.

Évaluation :
Rapport : Un rapport contenant la configuration des serveurs et des captures d’écran, à déposer sur Ametice avant le 27 avril.

Démo en TP : Présentation en salle de TP prévue le 28 et 29 avril.
